# Task Manager API

RESTful Task Management API built using Node.js and Express with automated API testing.

## Features
- User authentication
- CRUD operations for tasks
- MongoDB-based persistence
- Automated API tests using Keploy
- CI-ready structure

## Tech Stack
- Node.js, Express
- MongoDB
- Keploy (API testing)
- GitHub Actions (optional)

## API Endpoints
- POST /auth/register
- POST /auth/login
- GET /tasks
- POST /tasks
- PUT /tasks/:id
- DELETE /tasks/:id
